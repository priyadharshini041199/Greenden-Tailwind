# Greenden-Tailwind
Greenden

Greenden is a modern, responsive web project built with Tailwind CSS, designed to showcase clean design, smooth navigation, and a user-friendly interface. This project demonstrates the power of utility-first CSS for building stylish and functional web pages quickly and efficiently.

Features

Fully responsive design for mobile, tablet, and desktop

Lightweight and fast-loading for better performance

Easily customizable with Tailwind CSS utilities

Ideal for learning front-end development or showcasing web design skills

Technologies Used

HTML5

Tailwind CSS
